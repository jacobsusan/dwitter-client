DWITTER-CLIENT
==============

Framework : React

Command : npm start
URL : http://localhost:3000/dwitter

Features:
---------

1. Sign Up a Dweeter

2. Log in

3. View all the Dweets from all Dweeter followed by you

4. Make a Dweet

5. Search Dweeter

6. Follow a Dweeter

7. Search a Dweet

8. Like a Dweet

9. Comment on a Dweet

10. View the likes on a dweet

11. View the comments on a dweet

12. Log out
